## Project Config ⏳ Use Xcode Rename (Safest)

Close Xcode, then either:

**Option A: Xcode GUI**
1. Open `MeetRecorder.xcodeproj`
2. Click the project name in the navigator → press Enter → type `Glyph`
3. Xcode will offer to rename targets, schemes, and folders
4. Verify the Product Name build setting becomes `Glyph`

**Option B: project.yml + xcodegen**
1. Update `project.yml`:
   - `name: Glyph`
   - `PRODUCT_NAME: Glyph`
   - All `INFOPLIST_KEY_*UsageDescription` strings
2. Run `xcodegen generate`

## Bundle Identifier ⏳ Update Once
